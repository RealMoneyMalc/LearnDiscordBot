How To Make A Discord Bot

Here, you will learn how to very easily create a discord bot with Python. There are no downloads required, and the code is easy to understand for those who already have some background in python.

The only tools you require are the internet. Since this can be done fully online, you can do this on any device that can access the internet (yes even a smart fridge).

Firstly, visit to the Discord Developer Portal [link] and navigate to "Applications". From there you will see a blue button in the top-right of your screen that says "New Application". Click the button and make a name for your bot, and then click "Create" just below.

Now, go to the "Bot" tab listed on the left and select "Add Bot" once you reach that page. After confirmation, your bot has been created.

Now, you can go to Repl.it.com and create a new project in Python. On lines 1 and two in main.py, type in:
import discord
import os
this imports the discord.py library and the os library.
Then type in:
client = discord.Client()
This creates a Discord client, which is the connection to Discord. This sends the commands you create to Discord's servers.
After this, type in:
@client.event
async def on_ready():
print('We have logged in as {0.user}'.format(client))


@client.event
async def on_message(message):
if message.author == client.user:
return
'@client.event' is used to register an event happening. The event here is 'on_ready()', which tells the bot when to start functioning. Another event is the 'on_message()', which executes when the bot receives a message.
Next, type in:
if message.content.startswith('!c'):
await message.channel.send('You are using a command')
This area defines exactly what we want the bot to do. Here is where you can customize types of messages and what phrases you want the bot to respond to.
Lastly, type in:
discordtoken = os.environ.get("TOKEN")
client.run(discordtoken)
These last couple of lines get the specific token for your bot.
All the code combined should look like this:
import discord
import os
client = discord.Client()


@client.event
async def on_ready():
print('We have logged in as {0.user}'.format(client))

@client.event
async def on_message(message):
if message.author == client.user:
return

if message.content.startswith('!c'):
await message.channel.send('You are using a command')

discordtoken = os.environ.get("TOKEN")
client.run(discordtoken)
Now, you need to set up a file called .env to correlate with the last two lines in main.py. On Repl.it, next to the word 'Files' in the top left of the webpage, there are two icons: a portrait-facing paper with a plus and a landscape-facing folder with a plus. Click the portrait-facing paper and name the new file .env (don't forget the period).

In this file, type:
TOKEN=
After the equals sign is where you will put your bot token. To get this token, navigate back to the Discord Developer Portal to the Bot tab. Next to or underneath the large icon/profile picture of your bot, there are two blue buttons that say 'Copy' and 'Regenerate'. Click the 'Copy' button. You have copied your token. Go back to your .env file in Repl.it and paste the token next to the equals sign.

Now go back to the Discord Developer Portal and go to the OAuth2 tab, which is right above the Bot tab. Here you can choose permissions for your bot, and once you finish choosing your permissions, click the blue 'Copy' button on top of the Permissions module and paste the URL into your browser.

You will be asked to Authorize the bot and choose a server. Once you do, your bot is now linked to your server and can be used.

Go back to Repl.it and click the green 'RUN' button at the top center of the webpage and after a few seconds, your bot should be ready.

Now go to the discord server you linked the bot to and type !c. The bot should respond with, You are using a command.

Feel free to customize your bot further after this.
